

# Ratul Hasan Lemon — Portfolio Website

## Overview
A dark, premium personal portfolio website with neon green (#CCFF00) accents, smooth animations, and a "Command Center" aesthetic. Single-page layout with smooth scrolling between sections.

---

## Sections & Features

### 1. Sticky Navigation
- Dark blurred header with nav links: Home, About, Services, Skills, Portfolio, Experience, Contact
- Active section highlighted with neon green dot
- Mobile hamburger menu

### 2. Hero Section
- **Left:** Animated headline "Creative Digital Designer & Tech Specialist" with staggered text entrance. Service tags (Graphic Design • Social Media • Cyber Security • Web Dev • Video Editing) appearing one by one. Three CTA buttons: "View Portfolio", "Download CV", "Contact Me" with neon fill-up hover effect
- **Right:** Profile photo (uploaded image) with gradient mask at bottom, subtle floating animation, and radial neon glow behind it
- **Background:** Dark gradient (#0A0A0A) with subtle abstract grid/dot pattern

### 3. About Me Section
- Left-aligned text block with the provided bio content
- Fade-in-up animation on scroll

### 4. Services Section
- 3-column grid of animated cards (dark surface #141414, neon border on hover)
- 6 service cards: Graphic Design, Social Media Management, Cyber Security, Web Coding, Video Editing, Branding & Identity
- Each with a Lucide icon, title, short description, and hover glow effect

### 5. Skills Section
- Skill items displayed as animated progress bars with percentage labels
- Skills: Branding, Logo Design, Poster Design, Social Media Design, Web Banner, Cyber Security, Web Coding, Video Editing, Premiere Pro, After Effects, CapCut
- Monospace font (JetBrains Mono) for labels

### 6. Portfolio Section
- Bento-style 2-column grid with category filter pills (Branding, Poster, Social Media, Web Banner, Advertising, Packaging)
- Placeholder project cards with hover zoom + blur overlay showing project title
- External links to Behance and Facebook portfolio pages

### 7. Experience Timeline
- Vertical timeline with neon-glowing nodes
- 6 entries from CV: R10 Creative Co., Dainik Jonotar Khobor, Momento Dhaka, Munshiganj IT, TeachBD21, Gadget Waki
- Dates in monospace, descriptions in body font

### 8. CV Download Section
- Centered CTA section with download button linking to the uploaded PDF

### 9. Contact Section
- Contact info (email, phone, location) + contact form (name, email, message)
- WhatsApp button linking to +8801537543335
- Social media icon links (Behance, Facebook)

### 10. Footer
- Social links (Behance, Facebook, Email, Phone)
- "© 2026 Ratul Hasan Lemon — Creative Graphic Designer & Digital Specialist"

## Design System
- **Background:** #0A0A0A | **Surface:** #141414 | **Accent:** #CCFF00 (neon green)
- **Fonts:** General Sans (display), IBM Plex Sans (body), JetBrains Mono (labels) via Google Fonts
- **Animations:** Fade-in-up on scroll, staggered entrances, hover scale/glow effects, floating profile image
- **Fully responsive** with mobile-first approach
